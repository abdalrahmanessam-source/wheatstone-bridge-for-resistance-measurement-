# Wheatstone Bridge for Resistance Measurement

This project demonstrates the use of a Wheatstone Bridge circuit to measure an unknown resistance accurately.

## Overview
The Wheatstone Bridge is an electrical circuit used to determine an unknown resistance by balancing two legs of a bridge circuit. It is widely used in measurement systems and sensor applications because of its accuracy and sensitivity.

## Objective
- Measure an unknown resistance using the Wheatstone Bridge principle
- Understand bridge balancing conditions
- Analyze the relationship between known and unknown resistors

## Circuit Description
The Wheatstone Bridge consists of:
- Two known resistors
- One variable or standard resistor
- One unknown resistor
- A voltage source
- A galvanometer or detector

The bridge is said to be balanced when no current flows through the galvanometer.

## Working Principle
At balance condition:

R1 / R2 = R3 / Rx

Where:
- `R1` and `R2` are known resistors
- `R3` is a variable or known resistor
- `Rx` is the unknown resistance

Thus, the unknown resistance can be calculated as:

Rx = (R2 × R3) / R1

## Components Used
- Resistors
- DC voltage source
- Galvanometer / Multimeter
- Breadboard / simulation tool
- Connecting wires

## Procedure
1. Set up the Wheatstone Bridge circuit.
2. Connect the unknown resistor in one arm of the bridge.
3. Adjust the variable resistor until the galvanometer reads zero.
4. Note the resistor values at balance.
5. Calculate the unknown resistance using the balance equation.

## Applications
- Measuring medium resistances accurately
- Sensor circuits
- Strain gauge measurement
- Instrumentation systems

## Advantages
- High accuracy
- Sensitive to small resistance changes
- Simple and reliable circuit design

## Example Calculation
If:
- R1 = 100 Ω
- R2 = 200 Ω
- R3 = 150 Ω

Then:

Rx = (200 × 150) / 100 = 300 Ω

So, the unknown resistance is `300 Ω`.

## Conclusion
The Wheatstone Bridge is an effective method for accurately measuring unknown resistance. It is especially useful in electrical and instrumentation applications where precision is important.
